- Installer never overwrites existing user config (safe merge)
- Context preservation enforced via MCP server `context-keeper` (src/mcp/context-keeper.ts)
- context-keeper auto-registered to `opencode.json` (NOT mcp.json) via installer + migration
- `config/mcp.json` = Claude Desktop format (legacy); OpenCode reads `opencode.json` key `"mcp"`
- Backup enforcement: TodoWrite completion = context update (IRON RULE in AGENTS.md)
- Auto-Dispatch in AGENTS.md: AI silently selects agent + loads skills per message
- `opencode-jce update` backs up AGENTS.md before overwriting
- LSP auto-detect: installer scans PATH for all lsp.json commands, pre-populates opencode.json
- context-keeper responses include REMINDER lines to enforce AI compliance
- loadOpenCodeConfig() auto-creates full template if file missing (never produces partial config)
- install.ps1 uses [System.IO.File]::WriteAllText with UTF8NoBOM (PS 5.1 compat)
- `opencode-jce update` now also updates ~/.config/opencode/cli/ folder (not just global binary)
